# Adaptions-Patterns

Wiederkehrende Muster bei der Übersetzung von Literatur in Film. Jedes Pattern dokumentiert:
- **Problem**: Was im Roman steht und filmisch nicht direkt übersetzbar ist
- **Lösung**: Wie es filmisch gelöst wird
- **Beispiel**: Konkreter Fall aus einem Projekt
