# Story Values — McKees Wertesystem

#konzept #mckee #story-values #dramatik #szenenanalyse

**Story Values** sind das Herzstück von Robert McKees Dramaturgietheorie. Sie sind die Grundlage für alles, was eine Szene, eine Sequenz und einen Film dramatisch macht. Ohne ein Verständnis der Story Values versteht man nicht, warum manche Szenen funktionieren und andere nicht.

---

## Was sind Story Values?

Story Values sind **menschliche Erfahrungsdimensionen**, die positiv oder negativ geladen sein können — und die sich im Verlauf einer Geschichte verändern.

McKee definiert: **Story Values sind die universellen Qualitäten menschlicher Erfahrung, die vom einen in den anderen Zustand wechseln können.**

Beispiele:
- **Leben ↔ Tod**
- **Liebe ↔ Hass** (oder: Liebe ↔ Gleichgültigkeit)
- **Freiheit ↔ Knechtschaft**
- **Wahrheit ↔ Lüge**
- **Mut ↔ Feigheit**
- **Stärke ↔ Schwäche**
- **Loyalität ↔ Verrat**
- **Hoffnung ↔ Verzweiflung**
- **Würde ↔ Demütigung**
- **Gerechtigkeit ↔ Ungerechtigkeit**

---

## Das Grundprinzip: Jede Szene verändert einen Wert

McKees fundamentalste Regel:

> **Eine Szene ist eine dramatische Handlung, durch die ein Charakter in Konflikt eine Situation verändert, die zumindest einen der Werte des Lebens in einer signifikanten Weise vom positiven zum negativen oder vom negativen zum positiven Pol dreht.**

Wenn eine Szene mit derselben Werteladung endet, mit der sie begann — wenn sich nichts verändert hat — ist die Szene überflüssig und muss gestrichen oder umgeschrieben werden.

### Das binäre Grundschema

```
Positiv  (+)  ←→  Negativ  (−)
Leben         ←→  Tod
Liebe         ←→  Hass
Freiheit      ←→  Knechtschaft
Sieg          ←→  Niederlage
```

### Die ironic charge (ironische Ladung)

McKee betont, dass die interessantesten Verschiebungen nicht einfach von + nach − oder von − nach + gehen, sondern **ins Ironic** — in einen Zustand, der beide Pole gleichzeitig enthält oder der das Gegenteil von dem ist, was der Protagonist beabsichtigt hat.

Beispiel: Ein Protagonist gewinnt etwas (positive Werteladung: Sieg), verliert dabei aber etwas Wesentlicheres (negative Ladung: Verlust der Würde). Das Endergebnis ist **ironic**: weder rein positiv noch rein negativ, sondern ein Zustand, der komplexer ist als beide Pole.

## Positive und Negative Ladung

### Positive Ladung
Wenn eine Szene mit einem Wert auf der positiven Seite endet:
- Der Held gewinnt Kontrolle
- Eine Wahrheit wird enthüllt
- Eine Verbindung wird gestärkt
- Freiheit wird erlangt

Positive Ladungen fühlen sich für die Figur gut an — aber für den Zuschauer können sie sowohl Erleichterung als auch Unruhe erzeugen (falsche Sicherheit).

### Negative Ladung
Wenn eine Szene mit einem Wert auf der negativen Seite endet:
- Kontrolle geht verloren
- Eine Lüge wird nicht enthüllt
- Eine Verbindung zerbricht
- Freiheit wird genommen

**Wichtig:** Negative Ladung ist nicht dasselbe wie "schlechte" Dramatik. Eine negative Verschiebung kann die intensivste und befriedigendste dramatische Erfahrung sein — wenn sie die logische Konsequenz von Charakter und Handlung ist.

## Wertverschiebungen in einer Szene

McKees Szenen-Analyse arbeitet mit der Frage: **Von welcher Werteladung zu welcher?**

### Schematische Darstellung einer Szene

```
BEGINN DER SZENE          ENDE DER SZENE
Wert: Sicherheit (+)  →   Wert: Bedrohung (−)
Wert: Einigkeit (+)   →   Wert: Misstrauen (−)
Wert: Hoffnung (+)    →   Wert: Verzweiflung (−)
```

Oder in umgekehrter Richtung:

```
Wert: Isolation (−)   →   Wert: Verbundenheit (+)
Wert: Lüge (−)        →   Wert: Wahrheit (+)
Wert: Ohnmacht (−)    →   Wert: Handlungsfähigkeit (+)
```

### Mehrere Werte gleichzeitig
In einer Szene können mehrere Werte gleichzeitig in Bewegung sein, aber **ein Wert dominiert** die Szene und bestimmt ihre dramatische Bedeutung.

## Das Wertehierarchie-Prinzip

McKee unterscheidet zwischen verschiedenen Ebenen von Werten:

### Oberflächenwerte (Bewusste Ziele)
Was der Protagonist bewusst will. Diese Werte sind meist konkret und szenenspezifisch.

Beispiel: Ein Detective will ein Geständnis aus dem Verdächtigen herausbekommen.
→ Oberflächenwert: Erfolg/Misserfolg

### Tiefenwerte (Wahre Stakes)
Was wirklich auf dem Spiel steht. Diese Werte sind abstrakter und tiefer.

Beispiel: Kann Wahrheit gegen Macht bestehen?
→ Tiefenwert: Gerechtigkeit/Korruption

Die besten Szenen operieren auf beiden Ebenen gleichzeitig: Der Protagonist kämpft um ein konkretes Ziel (Oberflächenwert), aber das Publikum spürt, dass es um etwas Grundsätzlicheres geht (Tiefenwert).

---

## Story Values im Kontext der Geschichte (Thematic Value)

Das **zentrale Story Value** einer Geschichte definiert ihr Thema.

McKee: Jede Geschichte kreist um ein fundamentales Wertepaar. Dieses Wertepaar bestimmt:
- Was im Climax auf dem Spiel steht
- Welche Art von Antagonismus die Geschichte antreibt
- Welche emotionale Erfahrung das Publikum macht

**Beispiele für Kern-Story-Values:**

| Genre/Typ | Zentrales Wertepaar |
|---|---|
| Tragödie | Leben/Tod |
| Liebesgeschichte | Liebe/Hass oder Verbundenheit/Isolation |
| Politischer Thriller | Gerechtigkeit/Tyrannei |
| Coming-of-Age | Abhängigkeit/Autonomie |
| Horrorfilm | Sicherheit/Terror (Leben/Tod im emotionalen Sinne) |
| Kriegsfilm | Mut/Feigheit oder Ordnung/Chaos |

## Szenen-Analyse-Prozess nach McKee

**Schritt-für-Schritt-Analyse einer Szene:**

1. **Story Value identifizieren:** Was ist der dominante Wert dieser Szene? (Nicht: Was passiert? Sondern: Was steht auf dem Spiel?)

2. **Startladung bestimmen:** Wie ist der Wert zu Beginn geladen? Positiv oder negativ?

3. **Wendepunkt finden:** In welchem Moment dreht sich die Ladung? Welcher Beat löst die Veränderung aus?

4. **Endladung bestimmen:** Wie ist der Wert am Ende geladen?

5. **Subtext prüfen:** Was sagen die Figuren im Text? Was meinen sie im Subtext? Sind Text und Subtext entgegengesetzt? (Das ist McKees Ideal.)

6. **Szenen-Funktion beurteilen:** Hat sich der Wert verändert? Wenn nein: Szene streichen oder umschreiben.

---

## Positiv/Negativ im Verhältnis zum Protagonisten

Ein wichtige Nuance: "Positiv" und "Negativ" bedeuten bei McKee nicht "gut" und "schlecht" im moralischen Sinne, sondern werden aus der Perspektive des **Protagonisten und seiner Ziele** definiert.

Was für den Protagonisten positiv ist (Triumph, Sicherheit), kann für andere Figuren negativ sein. Was für den Antagonisten gut ist, ist für den Protagonisten negativ. Dies schafft die **moralische Komplexität** echter Dramatik.

McKee betont: Die besten Geschichten haben Antagonisten, die aus ihrer eigenen Perspektive vollkommen Recht haben. Ein Antagonist, der "böse" ist, weil er böse sein will, ist flach. Ein Antagonist, der dasselbe fundamental menschliche Bedürfnis verfolgt wie der Protagonist (Gerechtigkeit, Liebe, Überleben), schafft das tiefste Drama.

---

## Ironic Charge — Die fortgeschrittenste Werteverschiebung

McKee beschreibt die **ironic charge** als das Kennzeichen großen Schreibens:

> Eine Szene endet nicht einfach mit einer umgekehrten Werteladung, sondern mit einem Zustand, der das Wertesystem selbst in Frage stellt.

Beispiel: Ein Protagonist befreit sich aus einer Gefangenschaft (Wert: Freiheit → positiv). Aber in der Freiheit erkennt er, dass er seine Identität verloren hat (Wert: Identität → negativ). Das Ironic ist: Freiheit hat ihn unfrei gemacht in einem tieferen Sinne.

Diese ironic charges sind der Grund, warum manche Filme noch Jahre nach dem Sehen nachwirken: Sie hinterlassen den Zuschauer in einem Zustand der produktiven Ambivalenz.

## Anwendungsregel für Adaptation

### Werte des Quellromans identifizieren

1. **Zentrales Wertepaar des Romans benennen:** Was ist das Thema auf der tiefsten Ebene? (McKee-Frage: "What does this story prove about human nature?")

2. **Szenenweise Wertekarte erstellen:** Für jede Schlüsselszene des Romans: Startwert (+/−) → Endwert (+/−). Notieren, welche Szenen "flach" sind (keine Veränderung).

3. **Flache Szenen bewerten:** Hat die Szene narrativen Zweck (Exposition), aber keinen dramatischen? Wenn nein: weglassen. Wenn ja: Wie kann ein Wertekonflikt eingebettet werden?

4. **Subtext entwickeln:** Romane können inneren Konflikt direkt zeigen. Im Drehbuch muss dieser Konflikt in **Verhalten** und **Dialog** übersetzt werden. Die Figuren sagen, was sie nicht meinen — der Subtext ist der eigentliche Text.

5. **Ironic Charges bewahren:** Wenn der Roman ambivalente, komplexe Werteverschiebungen hat (nicht einfach gut/schlecht), diese nicht in der Adaptation vereinfachen. Sie sind oft das Wertvollste.

### Typische Adaptions-Fallen bei Werten

- **Vereinfachung:** Roman hat ironic charge; Adaptation macht klares Happy End. Lösung: Ironic charge bewahren.
- **Werteverlust durch Kürzung:** Eine Nebenhandlung wird gestrichen; mit ihr geht ein wichtiger Wert verloren. Lösung: Prüfen, ob der Wert andernorts eingebaut werden kann.
- **Falsche Gewichtung:** Oberflächenwert (Was will der Held?) wird betont; Tiefenwert (Was steht wirklich auf dem Spiel?) wird vernachlässigt. Lösung: Tiefenwert explizit in Szenen übersetzen.

## Story Values und Snyders Genre-System

Snyders 10 Genres (→ [[blake-snyder]]) lassen sich durch McKees Story Values beschreiben:

| Snyder-Genre | Dominantes Wertepaar (McKee) |
|---|---|
| Monster in the House | Leben/Tod, Sicherheit/Terror |
| Golden Fleece | Selbstkenntnis/Unwissenheit, Wachstum/Stagnation |
| Out of the Bottle | Authentizität/Illusion, Verdienst/Unrecht |
| Dude with a Problem | Überleben/Tod, Würde/Demütigung |
| Rites of Passage | Autonomie/Abhängigkeit, Reife/Unreife |
| Buddy Love | Verbundenheit/Isolation, Liebe/Einsamkeit |
| Whydunit | Wahrheit/Lüge, Gerechtigkeit/Korruption |
| Fool Triumphant | Würde/Demütigung, Vernunft/Torheit |
| Institutionalized | Identität/Konformität, Freiheit/Einschränkung |
| Superhero | Akzeptanz/Ablehnung, Stärke/Einsamkeit |
