# Adaptions-Brain — Drehbuch-Wissensnetz

Dieses Vault ist das **projektübergreifende Drehbuch-Gehirn** für alle Literaturadaptionen. Es enthält destilliertes Wissen aus 8 Theorie-Büchern, 20 Adaptions-Drehbüchern und 48 professionellen Screenplays — vernetzt als navigierbares Wissensnetz.

**Stand:** 54 Notes, 360+ Wikilinks. **Lernagenda T2.1–T2.6 komplett abgeschlossen** 2026-04-23: T2.1 Adaptionstheorie (Seger/McKee + Figurenfusion + Plot-Umstrukturierung), T2.2 Tolstoi-Profil + Erzählstil, T2.3 HM-Werkanalyse (projektspezifisch), T2.4 Kaukasus-Kontext + Transliteration, T2.5 fünf Roman-zu-Film-Fallstudien, T2.6 vier Strukturmethoden (Framing, Intercutting-Struktur, Non-Linear, Antiheroische Fokalisierung). Seger-Volltext Kap. 1 + 4–8 als OCR-Quelle.
