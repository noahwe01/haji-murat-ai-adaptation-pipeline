# Screenplay-Formatting – Drehbuch-Industriestandard

Formatiere adaptierte Szenen und Drehbücher nach professionellem Industriestandard.

## Output-Format: Fountain

Die Pipeline produziert Drehbücher im **Fountain-Format** (Plain-Text, human-readable, auto-renderbar in Final-Draft/Highland). Das unten stehende Layout beschreibt, wie Fountain die einzelnen Screenplay-Elemente rendert.

## Grundregeln

- **Rendering-Target**: Courier 12pt (1 Seite ≈ 1 Minute Screentime)
- **Sprache**: Präsens, aktive Sätze, visuelle Beschreibung, **englisch** (Pipeline-Output-Konvention)
- **Seitenzahl**: Feature 90–120 Seiten

## Die Elemente

### 1. Scene Heading / Slugline
```
INT. CHESS CLUB - NIGHT
EXT. COURTYARD - DAY
```
- INT. (Interior) oder EXT. (Exterior)
- Ort in Großbuchstaben
- DAY, NIGHT, DAWN, DUSK, CONTINUOUS

### 2. Action / Szenenbeschreibung
```
A figure sits alone at the board. Fingers hover over the knight,
trembling. Silence around them.
```
- Präsens, dritte Person
- Nur beschreiben, was man **sehen und hören** kann
- Kurze Absätze (max. 4 Zeilen)
- Neue Figuren beim ersten Auftritt in GROSSBUCHSTABEN

### 3. Character Name
```
                         CHARACTER
```
- Zentriert, Großbuchstaben
- Über dem Dialog

### 4. Parenthetical
```
                    (quietly, not looking up)
```
- Sparsam verwenden – nur wenn Tonfall nicht aus Kontext klar

### 5. Dialog
```
              I'm not sacrificing the knight. I'm
              sacrificing myself.
```
- Zentriert unter dem Character Name
- Knapp, subtextreich, charakterspezifisch

### 6. Transitions
```
                                          CUT TO:
                                          FADE OUT.
```
- Rechtsbündig, sparsam verwenden
- CUT TO / SMASH CUT / FADE OUT

### 7. Besondere Elemente
- **MONTAGE**: Reihe kurzer Szenen ohne Dialog
- **INTERCUT**: Parallele Szenen (z. B. Telefonat)
- **V.O.** (Voice Over): Erzählerstimme aus dem Off
- **O.S.** (Off Screen): Figur spricht, ist aber nicht im Bild
- **SUPER**: Text-Einblendung (Ort, Zeit)

## Adaptions-Tipp
Beim Übergang von Treatment zu Drehbuch: Jede Treatment-Passage in einzelne Szenen mit Sluglines aufbrechen. Eine Szene = ein Ort + eine Zeit. Ortswechsel = neue Szene.
