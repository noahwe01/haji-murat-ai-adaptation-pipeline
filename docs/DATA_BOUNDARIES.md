# Data boundaries

This repository is an intentionally curated export of one pipeline and its run artifacts.

## Included

- Framework and agent implementation
- Runtime skills and condensed knowledge notes
- Project configuration and public-domain source input
- State checkpoints and machine-readable analysis results
- Prompt and result captures from pipeline runs
- Obsidian project content
- Graphify inputs and outputs
- Adaptation decision logs

## Excluded

- Standalone creative deliverables and rendered documents
- Assessment and review documents
- Personal work notes and conversation handoffs
- Local application settings and caches
- Unrelated projects
- Third-party reference books, screenplays, treatments, extracted corpus payloads, and generated corpus index
- Credentials, secrets, and API keys

## Normalization applied to the export

- Absolute local paths were converted to repository-relative paths.
- Personal metadata and unrelated purpose labels were removed from technical artifacts.
- Runtime data was otherwise kept in its original structure.

## Important state detail

State snapshots may contain intermediate creative text in fields such as `treatment_text` or `final_script`. These fields are retained because they are part of the inter-phase data contract. Separate rendered or standalone deliverables are not included.
